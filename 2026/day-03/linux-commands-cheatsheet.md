1. Process Management

Command	      Purpose

ps -ef	      View all running processes

top	          Real-time process monitoring

htop	      Interactive process viewer (if installed)

pidof <process>	 Get PID of a process

kill <PID>	     Terminate a process

kill -9 <PID>	Force kill a process

pkill <name>	Kill process by name

jobs	        View background jobs

bg	            Run job in background

fg	            Bring job to foreground

systemctl status <service>	Check service status


2. File System Commands

Command	       Purpose

pwd	         Show current directory

ls -la	        List files with details

cd <dir>	    Change directory

mkdir <dir>	    Create directory

touch file.txt	Create file

cp source dest	Copy files

mv source dest	Move/Rename files

rm -rf <dir>	Delete files/directories

find /path -name file	  Search files

du -sh *	            Check directory sizes

df -h	             Check disk usage

cat file	View file content

less file	Read large files

chmod	    Change permissions

chown	    Change ownership



Command	  Purpose

ip a	View IP addresses

ip route	View routing table

ping <host>	Test connectivity

curl <url>	Test HTTP/HTTPS access

wget <url>	Download files

nslookup <domain>	DNS lookup

dig <domain>	Detailed DNS query

ss -tulpn	Check listening ports

netstat -tulpn	View network connections
